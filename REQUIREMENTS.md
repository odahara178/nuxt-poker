# テキサスホールデム ポーカーゲーム 要件定義書

## 概要

既存のドローポーカー実装をテキサスホールデムに作り直した1人用ポーカーゲーム。
対戦相手はAI 1体。チップ＝ポイント方式で、連勝ボーナスと役強さボーナスにより短時間でも楽しめる設計。

---

## ゲームルール

### 基本ルール
- テキサスホールデム方式（ホールカード2枚 + コミュニティカード5枚）
- 1人 vs AI 1体のヘッズアップ形式
- 初期チップ: 各1000枚
- スモールブラインド: 10（プレイヤー）
- ビッグブラインド: 20（AI）

### フェーズ進行
```
IDLE → startNewRound() → PREFLOP → FLOP → TURN → RIVER → SHOWDOWN → RESULT
```
- プリフロップ: ホールカード2枚配布、ブラインドポスト後にベッティング
- フロップ: コミュニティカード3枚公開
- ターン: コミュニティカード4枚目公開
- リバー: コミュニティカード5枚目公開
- ショーダウン: AIカード公開・勝敗判定
- リザルト: ボーナス計算・チップ配布

フォールド時はその場でショーダウン（フォールド側が負け）。

### ベッティング
- アクション: フォールド / チェック / コール / レイズ
- レイズ額: 固定40チップ
- 1フェーズあたり各プレイヤー最大1回のレイズ
- プレイヤーが先行、AIが後行
- AIがレイズした場合、プレイヤーはコールまたはフォールドのみ可能

## ポイント（チップ）システム

### 連勝ボーナス倍率
| 連勝数 | 倍率 |
|--------|------|
| 0 | ×1.0 |
| 1 | ×1.2 |
| 2 | ×1.5 |
| 3 | ×2.0 |
| 4以上 | ×2.5 |

### 役強さボーナス倍率
| 役 | 倍率 |
|----|------|
| ハイカード | ×1.0 |
| ワンペア | ×1.0 |
| ツーペア | ×1.1 |
| スリーカード | ×1.2 |
| ストレート | ×1.5 |
| フラッシュ | ×2.0 |
| フルハウス | ×3.0 |
| フォーカード | ×5.0 |
| ストレートフラッシュ | ×8.0 |
| ロイヤルフラッシュ | ×10.0 |

最終ペイアウト = floor(ポット × 連勝倍率 × 役倍率)

引き分け時はポットを折半して返却。連勝はリセット。

## AI 意思決定

### スコアリング
- プリフロップ: ホールカードのランク・ペア・スーテッド・コネクター考慮
- ポストフロップ: evaluateBestHand の rankIndex × 11 をスコアとして使用

### アクション閾値
| スコア | アクション |
|--------|-----------|
| > 75 | レイズ（未レイズなら）/ コール |
| > 40 | コール（ポットオッズが高すぎればフォールド）|
| > 20 | チェック（コスト0）/ フォールド |
| ≤ 20 | フォールド |

### ランダム偏差
- 8%: ブラフ（スコア+35）
- 15%: ヒーローフォールド（スコア-35）
